    Human usable service to quickly get informations about student.
    • Need to call a friend but you don't have his number?
    • Need to calculate your GPA?
    • Need to see someone's face?
    • Need to get informations but the intranet is down?
     » Try Peuple!

  • Web Service
    If you want to code your own tool that would need information and want it to be always up to date, you can use the web service from any language. It can provide many different data formats and is easily usable. » Documentation, sources

  • PHP Class
    If you want to code your own tool in PHP that would require information about users Ionis, you can use this PHP class. It does not require much and makes all the retrieval of informations for you. Please consider that you will have to manage the updating of informations by yourself. » Documentation, sources and download
